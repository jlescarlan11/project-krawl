## üìã Logo Overview

The Krawl logo represents a **breadcrumb trail** connecting cultural "Gems" (points of interest). The design symbolizes:
- **Journey & Discovery** - The trail path represents exploration
- **Connection** - Gems connected by a path show cultural continuity
- **Community** - Multiple points represent community-curated content
- **Movement** - The curved path suggests dynamic exploration

**Design Features:**
- **Clean & Minimal** - Solid circles without inner highlights for clarity
- **Consistent Line Weight** - Trail path uses consistent thickness across all sizes
- **Color Sequence** - Green ‚Üí Orange ‚Üí Yellow ‚Üí Green (represents cultural diversity)

## üìê Logo Specifications

### Minimum Sizes
- **Full Logo:** Minimum 64px height
- **Favicon:** 32x32px (minimum), 16x16px (browser)
- **App Icon:** 512x512px (recommended)

### Clear Space
- Maintain **1x the height of the largest Gem** as clear space around the logo
- Example: If Gem is 24px diameter, maintain 24px clear space minimum

### Scaling
- Logo scales proportionally
- Maintain aspect ratio: 1:1 (square format)
- ViewBox: `0 0 128 128`

## üîß Technical Specifications

### SVG Format
- All logos provided in SVG format
- Optimized for web use
- Scalable without quality loss
- ViewBox: `0 0 128 128` (except favicon: `0 0 32 32`)

### Line Specifications
- **Main Logos:** Trail path stroke-width: `6px` (128x128 viewBox)
- **Favicon:** Trail path stroke-width: `1.5px` (32x32 viewBox)
- **Consistency:** Line thickness is proportionally consistent across all sizes

### Design Elements
- **Gems:** Solid circles, radius 12px (main logos) or 3px (favicon)
- **Trail Path:** Curved path connecting all Gems with consistent stroke-width
- **No Inner Highlights:** Clean, minimal design matching favicon style

### PNG Export Recommendations
When exporting to PNG:
- **Full Logo:** 512x512px (high-res), 256x256px (standard), 128x128px (web)
- **Favicon:** 32x32px, 16x16px
- **App Icon:** 1024x1024px (iOS), 512x512px (Android)

### File Naming Convention
- `krawl-logo-[variation].svg`
- `krawl-logo-[variation]-[size].png`
- `krawl-favicon.svg` or `favicon.ico`

## üìù Export Instructions

### Using Inkscape (Recommended)

1. **Open SVG file** in Inkscape
2. **File ‚Üí Export PNG Image**
3. **Set dimensions:**
   - Width: 512px (or desired size)
   - Height: 512px (maintains aspect ratio)
4. **Click Export**
5. **Save as:** `krawl-logo-full-color-512.png`

### Using Figma

1. **Import SVG** or recreate in Figma
2. **Select logo**
3. **Right-click ‚Üí Export**
4. **Choose format:** PNG
5. **Set scale:** 1x, 2x, 3x (for different resolutions)
6. **Export**

### Using Online Converters

1. **Upload SVG file**
2. **Set output format:** PNG
3. **Set dimensions:** 512x512px (or desired)
4. **Convert and download**

## üéØ Favicon Implementation

### HTML Implementation

```html
<!-- Standard favicon -->
<link rel="icon" type="image/svg+xml" href="/logo/krawl-favicon.svg">

<!-- Fallback for older browsers -->
<link rel="icon" type="image/png" sizes="32x32" href="/logo/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/logo/favicon-16x16.png">

<!-- Apple Touch Icon -->
<link rel="apple-touch-icon" sizes="180x180" href="/logo/apple-touch-icon.png">

<!-- Android Chrome -->
<link rel="manifest" href="/logo/site.webmanifest">
```

### Next.js Implementation

Place favicon files in `/public` folder:
```
/public
  /favicon.ico
  /favicon.svg
  /apple-touch-icon.png
```

Next.js will automatically detect and use them.
