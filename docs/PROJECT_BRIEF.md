**Date:** November 15, 2025

## üéØ Main Objectives

### Primary Goal
Build and launch a community-driven Progressive Web App (PWA) that maps authentic Filipino culture through user-curated points of interest ("Gems") and guided trails ("Krawls").

### Core Objectives
1. **Develop & Launch MVP** - Deploy functional PWA with core features (authentication, Gem pinning, Krawl creation, Krawl Mode, offline capability)
2. **Seed Initial Content** - Populate Cebu City with 100+ Gems and 10+ Krawls across key districts
3. **Validate Core Loop** - Achieve 100-500 active users in Cebu City engaging with core features
4. **Gather User Feedback** - Collect qualitative and quantitative data to inform iteration
5. **Test Revenue Model** - Onboard 5-10 local Cebu businesses to validate freemium model
6. **Map Boundary Restriction** - Limit map view and content creation to Cebu City boundaries only

## üéØ Problem-Solution Fit

**Problem:** No platform effectively combines community-sourced local knowledge with structured, navigable experiences for discovering authentic Filipino culture.

**Solution:** Krawl provides a community-driven PWA that enables users to map cultural "Gems" and create guided "Krawls" with offline-capable, location-aware navigation.

---

## üó∫Ô∏è Launch Area: Cebu City Only

**Initial Launch Focus:** Cebu City, Philippines

**Map Boundary Restrictions:**
- Map view limited to Cebu City boundaries only
- Gem creation restricted to locations within Cebu City
- Krawl creation limited to trails within Cebu City
- Search and discovery features scoped to Cebu City area
- Location validation enforces Cebu City boundaries

**Rationale:**
- Focused launch allows for deeper content curation and community building
- Easier to validate core features with concentrated user base
- Better quality control with limited geographic scope
- Foundation for future expansion to other cities

**Future Expansion:**
- Metro Manila and other Philippine cities planned post-MVP
- Expansion based on user feedback and validation from Cebu City launch

## üìñ Glossary

**Key Terms:**
- **Gem** - A point of interest representing authentic Filipino cultural location or experience, created by users
- **Krawl** - A guided trail connecting multiple Gems in a specific sequence, created by users
- **Krawl Mode** - Location-aware guided experience that helps users follow a Krawl in real-time with turn-by-turn directions
- **Vouching** - Community quality control mechanism where users can vouch for the authenticity/quality of Gems or Krawls
- **PWA** - Progressive Web App - a web application that provides app-like experience with offline capabilities
