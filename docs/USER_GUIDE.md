## Getting Started

### 1. Sign In

**Using Google:**
1. Click "Sign In" button
2. Choose "Continue with Google"
3. Select your Google account
4. Allow Krawl to access your basic profile

**As Guest:**
- Browse maps and content without signing in
- Sign in required for: creating content, Krawl Mode, saving favorites

### 2. Onboarding (Optional)

First-time users see a 5-step onboarding:
1. Welcome & value proposition
2. Grant location permission (optional)
3. Learn about features
4. Discover how to contribute
5. Start exploring!

**You can skip onboarding** - access it later in Settings > Help

## Creating Content

### Creating a Gem

**Requirements:**
- Signed-in account
- Location within Cebu City

**Steps:**
1. Click "+" â†’ "Create Gem"
2. **Location:** Select on map
3. **Basic Info:** Name, category, description
4. **Photos:** Upload 1-5 photos
5. **Cultural Significance:** Explain importance
6. **Preview:** Review before submitting
7. **Submit:** Gem goes live immediately

**Tips:**
- Use clear, descriptive names
- Upload high-quality photos
- Explain cultural context
- Check for duplicates
- Save drafts for later

### Creating a Krawl

**Requirements:**
- Signed-in account
- 2-12 Gems selected

**Steps:**
1. Click "+" â†’ "Create Krawl"
2. **Basic Info:** Name, description, difficulty
3. **Select Gems:** Choose 2-12 Gems
4. **Order Gems:** Drag to reorder stops
5. **Cover Image:** Upload cover photo
6. **Preview:** Review trail on map
7. **Submit:** Krawl goes live

**Tips:**
- Create themed trails
- Consider walking distance
- Optimize route order
- Add context in description

## Community Features

### Vouching

**What is Vouching?**
- Community quality signal
- One vouch per user per Gem/Krawl
- Shows content authenticity

**How to Vouch:**
1. View Gem or Krawl
2. Click "Vouch" button
3. Confirmation appears
4. Remove vouch anytime

### Rating

**How to Rate:**
1. View Gem or Krawl
2. Click on stars (1-5)
3. Rating saves automatically
4. Update rating anytime

### Commenting

**How to Comment:**
1. View Gem or Krawl
2. Scroll to comments section
3. Type comment (max 500 characters)
4. Click "Post Comment"

**Comment Guidelines:**
- Be respectful
- Share constructive feedback
- No spam or promotional content
- Report inappropriate comments

## FAQ

**Q: Is Krawl free to use?**
A: Yes, completely free!

**Q: Do I need to sign in?**
A: No for browsing, yes for creating content and Krawl Mode.

**Q: Can I use Krawl offline?**
A: Yes! Download Krawls for offline use.

**Q: How accurate is location tracking?**
A: Accuracy depends on your device GPS (typically 5-20 meters).

**Q: Can I edit my created Gems?**
A: Yes, from your profile â†’ My Gems.

**Q: What if I find inappropriate content?**
A: Use the "Report" button on any Gem or Krawl.

**Q: Is my location data private?**
A: Location is only tracked during active Krawl Mode sessions. See Privacy Policy.

## Troubleshooting

### Location Not Working

1. Grant location permission
2. Enable location services on device
3. Try outdoors (better GPS signal)
4. Restart app

### Can't Sign In

1. Check internet connection
2. Try different browser
3. Clear browser cache
4. Contact support

### Krawl Mode Not Starting

1. Check pre-start checklist
2. Grant location permission
3. Check battery level
4. Ensure Krawl is downloaded (if offline)

### Content Not Loading

1. Check internet connection
2. Refresh page
3. Clear browser cache
4. Try different network
